# publish-checklist

## 公開前
- `trip\YYYY-MM-DD_slug\summary.md` を確認
- `trip-blog\posts\YYYY-MM-DD-slug.md` を確認
- `draft: true` のまま内容確認
- 自宅周辺情報が `福山市` 表記に丸められているか確認
- 写真候補と本文が対応しているか確認
- Gフォルダ保存済みか確認

## GitHub反映
- GitHub管理フォルダへ反映
- diff確認
- commit
- push

## Cloudflare Pages
- 自動デプロイ結果確認
- 公開URLで最終確認
