# travel-log-publish

掲載用に整理した旅ログのたたき台です。

## 方針
- 自分用に見返して役立つ記録を優先
- 旅行記 + 実用メモの両立
- 将来的にアフィリエイト導線を追加可能な構成

## 次にやること
1. 実際の旅行1件分をこの構成に当てはめる
2. 写真を入れる
3. 固有のサイト名や説明文を決める
4. GitHub / Cloudflare Pages に本番用として反映する

## 想定構成
- hero: サイトの目的
- latest: 最新の旅ログ
- template: 記録テンプレート
- policy: 運用方針 / アフィリエイト方針
